## 一、需求背景

当前 TripAgent 已完成基础架构搭建（Agent 编排、FastAPI 服务、React 前端、JWT 认证、PostgreSQL 持久化），但在用户体系与商业化方面存在明显缺口：

| 问题 | 现状 | 影响 |
|------|------|------|
| 🔓 游客身份缺失 | 聊天接口无认证要求，所有请求匿名处理 | 无法追踪用户行为，无法限制滥用 |
| 🎟️ 免费额度未落地 | 代码注释写了"1次/设备"但未实现，仅做了 IP 级全局限流 | 无成本控制，API 费用不可控 |
| 💰 付费体系缺失 | User 表有 role 字段但无 credits；无支付、无套餐 | 0 商业化能力 |
| 📝 聊天历史丢失 | 消息仅存前端内存，刷新即消失 | 用户误操作无法恢复，体验差 |
| 📄 计划无法导出 | 无任何复制/导出功能 | 用户只能截图保存，非常不专业 |
| 📊 额度不可见 | 前端无任何额度/使用量展示 | 用户对剩余次数无感知 |

本方案针对以上 6 项缺口，给出完整的改造设计。

### 阶段二：游客设备绑定 + 免费额度（核心业务，2-3天）

#### 3.2.1 设备指纹方案

**原理**：前端使用 `@fingerprintjs/fingerprintjs` 生成浏览器指纹（Canvas、WebGL、字体等），SHA-256 哈希后作为唯一标识。

**流程**：

```
用户打开页面 → fingerprintjs 生成指纹 → 存入 localStorage
    ↓
发送请求时自动注入 X-Device-Fingerprint header
    ↓
后端 DeviceFingerprintMiddleware 解析 → 查询/创建 device_sessions 记录
    ↓
CreditService 根据 device_id 查剩余额度 → 决定是否放行
```

**前端代码（核心逻辑）**：

```typescript
// frontend/src/hooks/useDeviceFingerprint.ts
import FingerprintJS from '@fingerprintjs/fingerprintjs';

let cachedFingerprint: string | null = null;

export async function getDeviceFingerprint(): Promise<string> {
  if (cachedFingerprint) return cachedFingerprint;
  
  // 优先使用 localStorage
  const stored = localStorage.getItem('device_fp');
  if (stored) {
    cachedFingerprint = stored;
    return stored;
  }
  
  // 首次生成
  const fp = await FingerprintJS.load();
  const result = await fp.get();
  cachedFingerprint = result.visitorId;  // 已是 hash
  localStorage.setItem('device_fp', cachedFingerprint);
  return cachedFingerprint;
}
```

**后端中间件**：

```python
# src/server/middleware.py 新增

class DeviceFingerprintMiddleware:
    """解析 X-Device-Fingerprint header 并注入到 request.state"""
    
    async def __call__(self, scope, receive, send):
        if scope["type"] == "http":
            headers = dict(scope.get("headers", []))
            fp = headers.get(b"x-device-fingerprint")
            scope["device_fingerprint"] = fp.decode() if fp else None
        await self.app(scope, receive, send)
```

#### 3.2.4 软限制策略（关键设计）

游客用完 1 次免费额度后的体验流：

```
游客超额 → 弹出 CTA 卡片 → [注册登录] [了解更多]
    ↓
登录后额度不足 → 黄色进度条预警 + Tooltip "额度已用完，本次为体验模式"
    ↓
生成完成后 → "本次计划为体验模式，注册/升级以保存和导出"
```

**设计原则**：
- 游客：硬限制，1 次后引导注册（注册即送 10 次试用额度）
- 注册用户：软限制，超额度仍可生成（透支），但导出/保存功能受限
- 新用户注册：赠送 10 次试用额度（`credit_transactions` 记录，type=earn）

```python
# src/server/services/export_service.py
import markdown
from docx import Document
from io import BytesIO
import json

class ExportService:
    """行程多格式导出服务"""

    def __init__(self, trip: Trip):
        self.trip = trip
        self.md_content = trip.raw_markdown or self._build_markdown()

    def _build_markdown(self) -> str:
        """从 itinerary_json 重建 Markdown，带前端风格排版"""
        # 以格式化、人类友好的方式重建
        lines = [
            f"# 🗺️ {self.trip.title}",
            f"**目的地**：{self.trip.city}  |  **天数**：{self.trip.days}天\n",
        ]
        for day_label, spots in self.trip.itinerary_json.items():
            lines.append(f"## {day_label}\n")
            for s in spots:
                lines.append(f"- ⏰ {s.get('time', '--')}  **{s['name']}**")
                if s.get('duration'):
                    lines.append(f"  预计游玩 {s['duration']}")
                if s.get('price'):
                    lines.append(f"  门票 ¥{s['price']}")
                if s.get('transport'):
                    lines.append(f"  {s['transport']} → {s.get('next_spot', '')}")
                lines.append("")
        return "\n".join(lines)

    def to_markdown(self) -> bytes:
        """返回 Markdown 文件的 bytes"""
        return self.md_content.encode("utf-8")

    def to_html(self) -> str:
        """Markdown → 带样式的 HTML"""
        md = markdown.Markdown(extensions=["tables", "fenced_code"])
        body = md.convert(self.md_content)
        return f"""<!DOCTYPE html>
<html><head><meta charset="utf-8">
<style>
  body {{ font-family: -apple-system, sans-serif; max-width: 800px; margin: 0 auto; padding: 2em; }}
  h1 {{ color: #1a1a2e; }} h2 {{ color: #16213e; margin-top: 2em; }}
  li {{ margin: 0.5em 0; line-height: 1.6; }}
</style></head><body>{body}</body></html>"""

    def to_pdf(self) -> bytes:
        """HTML → PDF（需要 weasyprint）"""
        from weasyprint import HTML
        return HTML(string=self.to_html()).write_pdf()

    def to_docx(self) -> bytes:
        """生成 Word 文档（需要 python-docx）"""
        doc = Document()
        doc.styles['Normal'].font.name = 'Microsoft YaHei'
        doc.add_heading(self.trip.title, 0)
        doc.add_paragraph(f"目的地：{self.trip.city}  |  {self.trip.days}天")
        for day_label, spots in self.trip.itinerary_json.items():
            doc.add_heading(day_label, level=1)
            for s in spots:
                p = doc.add_paragraph()
                p.add_run(f"⏰ {s['time']}  ").bold = False
                p.add_run(s['name']).bold = True
        buffer = BytesIO()
        doc.save(buffer)
        return buffer.getvalue()

    def to_plain_text(self) -> str:
        """生成无格式纯文本，供前端一键复制"""
        lines = [
            f"{self.trip.title}",
            f"目的地：{self.trip.city}  ·  {self.trip.days}天",
            f"{'─' * 40}",
        ]
        for day_label, spots in self.trip.itinerary_json.items():
            lines.append(f"\n{day_label}")
            for s in spots:
                lines.append(
                    f"  {s.get('time','')}  {s['name']}"
                    f"  {'🎫¥'+str(s['price']) if s.get('price') else ''}"
                )
        return "\n".join(lines)
```
