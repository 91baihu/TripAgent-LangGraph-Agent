## 📍 扩展1：高德地图 API 真实路线规划

### 需求
将原有的 mock 坐标 + 简化 Haversine 距离计算，升级为接入 [高德地图 Web API](https://lbs.amap.com/api/webservice/summary/) 的真实路线规划。

### 实现

**新增文件：** [src/tools/amap_service.py](../src/tools/amap_service.py)

核心类 `AmapService` 封装了以下高德 API 端点：

| 方法 | API 端点 | 功能 |
|------|----------|------|
| `geocode()` | `/v3/geocode/geo` | 地名 → 经纬度坐标 |
| `get_distance()` | `/v3/distance` | 驾车距离 + 耗时 |
| `plan_route()` | 组合调用 | 完整路线规划（含交通方式推荐） |
| `search_poi()` | `/v3/place/around` | 周边 POI 搜索（餐饮/住宿/景点） |
| `search_restaurants()` | 委托 `search_poi()` | 周边餐厅搜索 |
| `search_hotels()` | 委托 `search_poi()` | 周边酒店搜索 |

**降级策略（三层 fallback）：**

```
高德 API 调用
  ├── API Key 存在 + 网络正常 → 真实数据
  ├── API Key 未配置 → FALLBACK_COORDS 字典 + Haversine 公式
  └── API 超时/异常 → FALLBACK_COORDS 字典 + Haversine 公式
```

**环境变量：**
```bash
AMAP_API_KEY=your_amap_api_key  # 在 config/.env.example 中已预置
```

### 修改文件
- [src/tools/route.py](../src/tools/route.py) — 重写为调用 `amap_service.plan_route()`，输出格式升级为表格
- [src/tools/__init__.py](../src/tools/__init__.py) — 导出 `amap_service`

### 测试覆盖（5 项）
- `test_fallback_geocode` — 已知景点坐标查询
- `test_fallback_geocode_unknown` — 未知地点处理
- `test_haversine_distance` — 球面距离算法精度
- `test_plan_route_fallback` — 完整路线 fallback
- `test_plan_route_unknown_spot` — 错误输入处理

```
search_restaurants(city, near_spot, budget_level)
  ├── amap_service.available + near_spot 非空
```

**数据标注：**
- 高德数据标注 `🛰️高德地图`

## 🧠 扩展3：BGE Embedding 向量检索升级

### 需求
将 HybridRetriever 从纯关键词 TF 匹配升级为 **BM25 关键词 + BGE 向量语义搜索** 的混合检索。

### 实现

**新增文件：** [src/rag/embeddings.py](../src/rag/embeddings.py)

`EmbeddingService` 自动检测可用的 embedding 后端：

| 优先级 | 后端 | 模型 | 说明 |
|--------|------|------|------|
| 1 | FlagEmbedding | BAAI/bge-small-zh-v1.5 | BGE 官方库，最快 |
| 2 | sentence-transformers | BAAI/bge-small-zh-v1.5 | 通用库，模型生态丰富 |
| 3 | TF-IDF | 无（字符级 n-gram） | 纯 Python，零依赖降级 |

**修改文件：** [src/rag/retriever.py](../src/rag/retriever.py)

升级内容：
1. **新增 `BM25Scorer` 类** — 实现标准 BM25 算法（k1=1.5, b=0.75），Uni-gram + Bi-gram 中文分词
2. **向量延迟初始化** — `_ensure_vectors()` 在首次检索时才加载模型
3. **混合评分** — `final_score = 0.3 × BM25 + 0.7 × vector_similarity`
4. **统计接口** — `stats()` 返回检索器状态（文档数、后端、权重）

**降级路径：**
```
HybridRetriever.search()
  ├── BGE 模型加载成功 → BM25 + 向量语义融合
  ├── BGE 加载失败/网络不通 → 纯 BM25
  └── 全部不可用 → TF-IDF 字符级匹配
```

### 测试覆盖（5 项）
- `test_backend_detection` — 后端自动检测
- `test_tfidf_encoding_fallback` — TF-IDF 降级模式
- `test_cosine_similarity` — 相似度算法正确性
- `test_batch_similarity` — 批量相似度
- `test_search_with_bm25` — BM25 检索基础功能

## 💰 扩展4：Token 用量追踪与成本估算

### 需求
实现对每次 LLM 调用的 token 计数和 DeepSeek API 成本估算。

### 实现

**新增文件：** [src/server/token_tracker.py](../src/server/token_tracker.py)

**核心组件：**

| 类 | 功能 |
|-----|------|
| `TokenUsage` | 单次调用的 token/cost 数据类 |
| `SessionStats` | 会话维度聚合统计 |
| `TokenTracker` | 全局追踪器（记录 + 统计 + 报告） |

**Token 计数优先级：**
```
count_tokens()
  ├── tiktoken 可用 → 精确 BPE tokenize
  └── tiktoken 未安装 → 字符估算
       ├── 中文字符 / 1.5
       └── 英文字符 / 4
```

**DeepSeek 定价模型（2024）：**

| 模型 | 输入价格 | 输出价格 |
|------|----------|----------|
| deepseek-chat | ¥1 / 1M tokens | ¥2 / 1M tokens |
| deepseek-reasoner | ¥4 / 1M tokens | ¥16 / 1M tokens |

**使用方法：**
```python
from server.token_tracker import token_tracker

# 记录调用
usage = token_tracker.record_call(
    model="deepseek-chat",
    input_tokens=500,
    output_tokens=200,
    session_id="session_123",
    user_id="user_456",
)

# 查看统计
print(token_tracker.format_cost_report(session_id="session_123"))
# 📊 Token 用量报告
# | 调用次数    | 5        |
# | 输入 Tokens | 2,500    |
# | 输出 Tokens | 1,200    |
# | 预估费用    | ¥0.0049  |

# 预算告警
if token_tracker.is_over_budget(user_id="user_456"):
    print("⚠️ 超过每日预算阈值")
```

**API 端点：**
- `GET /api/v1/usage` — JSON 用量数据
- `GET /api/v1/usage/report` — Markdown 格式报告

### Agent 集成
在 [src/agent/graph.py](../src/agent/graph.py) 的 `agent_node` 中，每次 LLM 调用后自动记录用量。

### 测试覆盖（6 项）
- `test_count_tokens_chinese` — 中文 token 估算
- `test_count_tokens_english` — 英文 token 估算
- `test_calculate_cost` — DeepSeek 定价精度
- `test_record_call` — 调用记录 + 会话统计
- `test_format_cost_report` — 报告格式化

## 🚀 使用指南

### 启用高德地图 API（可选）
```bash
# 在 .env 或 config/dev.env 中添加
AMAP_API_KEY=your_key_here
```
不配置则自动降级为本地 fallback。

### 启用 BGE Embedding（可选）
```bash
# 安装依赖后自动检测
pip install flagembedding  # 推荐
# 或
pip install sentence-transformers
```
未安装则自动降级为 TF-IDF 字符级匹配。

### 启用精确 Token 计数（可选）
```bash
pip install tiktoken
```
未安装则使用字符估算（误差约 ±20%）。
