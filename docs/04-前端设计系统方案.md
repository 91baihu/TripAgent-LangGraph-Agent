## 一、设计系统：Warm Editorial（暖调编辑风）

Design Demo V3 定义了一套完整的暖调编辑风格设计系统，灵感来自旅行杂志的排版美学。

### 1.1 色彩体系

| Token | 色值 | 用途 |
|-------|------|------|
| `ink` | `#1A1D20` | 主文字色 — 标题、正文 |
| `ink-secondary` | `#4A5568` | 次文字色 — 辅助说明 |
| `ink-tertiary` | `#8899AA` | 辅助文字 — 占位符、禁用态 |
| `teal` | `#0D9488` | 主色调 — 按钮、链接、高亮 |
| `teal-dark` | `#0F766E` | 深主色 — hover/active |
| `teal-light` | `#CCFBF1` | 浅主色 — 选中背景 |
| `rust` | `#E85D3F` | 强调色 — 警告、收藏、价格 |
| `rust-light` | `#FEF2EE` | 浅强调色 |
| `sand` | `#FBF7F2` | 暖色背景 — 卡片、页面底色 |
| `sand-dark` | `#F0EAE0` | 次级暖色 — hover、骨架屏 |
| `sand-bg` | `#E8E4DC` | 页面全局背景 |
| `warm-border` | `#E8E3DD` | 边框色 |
| `warm-border2` | `#F2EFEB` | 次级边框 |

**Tailwind 配置**：所有 token 已映射到 [tailwind.config.ts](../frontend/tailwind.config.ts) 的 `extend.colors` 中。

### 1.2 字体系统

| 层级 | 规格 | Tailwind Class | 用途 |
|------|------|---------------|------|
| H1 | 24px / Bold 700 / 32px | `text-h1` | 页面标题 |
| H2 | 20px / SemiBold 600 / 28px | `text-h2` | 卡片标题、弹窗标题 |
| H3 | 17px / Medium 500 / 24px | `text-h3` | 列表标题、行程天标题 |
| Body | 15px / Regular 400 / 22px | `text-body` | 正文、对话气泡 |
| Caption | 13px / Regular 400 / 18px | `text-caption` | 辅助说明、时间戳 |
| Small | 11px / Regular 400 / 16px | `text-small` | 标签、徽标 |

- **无衬线**：`Inter, PingFang SC, Microsoft YaHei, sans-serif`
- **衬线标题**：`Noto Serif SC, STSong, SimSun, serif` → 用于 Hero 标题、城市名、温度大数字
- **加载方式**：Google Fonts CDN（`index.html` 中引入）

### 1.3 间距体系（8px 栅格）

| Token | 值 | 用途 |
|-------|-----|------|
| `1` | 4px | 紧密元素间距（图标与文字） |
| `2` | 8px | 同类元素间距 |
| `3` | 12px | 相关但不同类间距 |
| `4` | 16px | 卡片内边距 |
| `5` | 20px | 段落间距 |
| `6` | 24px | 模块间距、页面边距 |
| `8` | 32px | 大模块间隔 |
| `12` | 48px | 页面上下安全区 |

### 1.4 圆角体系

| Token | 值 | Tailwind Class | 场景 |
|-------|-----|---------------|------|
| `xs` | 6px | `rounded-xs` | 标签、徽标 |
| `button` | 10px | `rounded-button` | 按钮、输入框 |
| `card` | 12px | `rounded-card` | 常规卡片 |
| `card-lg` | 16px | `rounded-card-lg` | 大卡片（行程卡片） |
| `bubble` | 18px | `rounded-bubble` | 对话气泡 |
| `modal` | 20px | `rounded-modal` | 弹窗 |
| `xl` | 32px | `rounded-xl` | 超大圆角 |

### 1.5 阴影体系

| Token | 值 | Tailwind Class |
|-------|-----|---------------|
| `warm-sm` | `0 1px 3px rgba(26,29,32,0.06)` | `shadow-warm-sm` |
| `warm-md` | `0 4px 16px rgba(26,29,32,0.08)` | `shadow-warm-md` |
| `warm-lg` | `0 12px 32px rgba(26,29,32,0.10)` | `shadow-warm-lg` |
| `card` | `0 2px 8px rgba(0,0,0,0.04)` | `shadow-card` |
| `card-hover` | `0 4px 12px rgba(0,0,0,0.06)` | `shadow-card-hover` |

### 1.6 动画系统

| 动画 | 时长 | 缓动 | Tailwind Class | 用途 |
|------|------|------|---------------|------|
| `fade-up` | 0.4s | cubic-bezier(0.4,0,0.2,1) | `animate-fade-up` | 卡片浮现 |
| `fade-in` | 0.3s | ease-out | `animate-fade-in` | 元素淡入 |
| `scale-in` | 0.3s | cubic-bezier(0.34,1.56,0.64,1) | `animate-scale-in` | 弹窗弹出 |
| `slide-right` | 0.3s | ease-out | `animate-slide-right` | 侧边滑入 |
| `shimmer` | 1.6s | ease-in-out | `animate-shimmer` | 骨架屏扫光 |
| `breathe` | 3s | ease-in-out | `animate-breathe` | Logo/图标呼吸 |
| `pulse-ring` | 2s | — | `animate-pulse-ring` | 聚焦态光环 |
| `dot-bounce` | 1.3s | ease-in-out | `animate-dot-bounce` | 思考中三点弹跳 |

天气粒子动效详见 [§3.2](#32-天气粒子动效)。

> 所有动效均支持 `prefers-reduced-motion` 降级。

## 三、天气卡片设计

WeatherCard 是前端最具创新性的组件，提供三种形态 + 纯 CSS 粒子动效。

### 3.1 三种显示模式

| 模式 | 使用场景 | 特征 |
|------|---------|------|
| **Hero** | 对话卡片内、独立展示 | 渐变背景 + 粒子动效 + 大温度数字(3.5rem) + 温度条 + 7日预报 |
| **Compact** | 地图右上角浮层 | Glassmorphism (`bg-white/85 backdrop-blur-xl`) + 单行温度 |
| **Inline** | 聊天气泡内 | 单行：icon + 温度 + 城市 + 天气 |

### 3.2 天气粒子动效

基于天气状况自动匹配的纯 CSS 动效（零 JS 依赖）：

| 效果 | 天气类型 | 实现 |
|------|---------|------|
| **SunParticles** | 晴 | 10 个光点，`sun-ray` 3s 动画，CSS 自定义属性控制漂移方向 |
| **RainDrops** | 雨/阵雨/雷暴 | 20 个雨滴，`rain-drop` 0.55-0.85s 下落动画 |
| **SnowFlakes** | 雪 | 15 个雪花，`snow-fall` 3-4.5s 下落 + 旋转动画 |
| **FogLayer** | 雾/霾 | 3 层模糊云团，`cloud-drift` 6s 水平漂移动画 |

所有动画定义在 [tailwind.config.ts](../frontend/tailwind.config.ts) 的 `extend.keyframes` 中。

### 3.3 天气 → 视觉映射

```typescript
// 16 种天气条件 → { 渐变背景, emoji, 粒子效果, 文字颜色, 中文标签 }
const WEATHER_THEMES = {
  "晴":   { bg: "weather-bg-sunny",  icon: "☀️", effect: "sun",   text: "text-amber-900", label: "晴朗" },
  "多云": { bg: "weather-bg-cloudy", icon: "⛅", effect: "cloud", text: "text-ink",      label: "多云" },
  "雨":   { bg: "weather-bg-rainy",  icon: "🌧️", effect: "rain",  text: "text-slate-700", label: "雨天" },
  "雪":   { bg: "weather-bg-snowy",  icon: "❄️", effect: "snow",  text: "text-slate-600", label: "下雪" },
  "雷":   { bg: "weather-bg-stormy", icon: "⛈️", effect: "storm", text: "text-white",    label: "雷暴" },
  "雾":   { bg: "weather-bg-foggy",  icon: "🌫️", effect: "fog",   text: "text-ink-secondary", label: "有雾" },
  // ... 共 16 种映射
};
```

### 3.4 温度进度条

彩色进度条，颜色随温度渐变：**冷蓝(0°C) → 舒适绿(15°C) → 暖橙(26°C) → 酷热红(34°C+)**。

7 种渐变背景定义在 [index.css](../frontend/src/index.css) 中（`.weather-bg-sunny/.rainy/.snowy/.stormy/.foggy/.cloudy/.night`）。

## 四、响应式规范

### 4.1 断点策略

| 断点 | 宽度 | 布局 | Tailwind |
|------|------|------|----------|
| 手机 | < 768px | 430px 居中框架 + 底部导航 + ViewSwitcher Tab 切换 | `default` |
| 平板 | 768-960px | 过渡布局 | `md:` |
| 桌面 | ≥ 960px | 左右分屏：聊天(flex-1) + 可视化面板(420px) | `md:` / `lg:` |

### 4.2 关键适配规则

```
ChatPage:
  手机：单栏 + BottomNav + ViewSwitcher（对话/地图/排名切换）
  桌面：左侧对话 + 右侧可视化面板（420px 固定宽度）

TripListPage:
  手机：单列卡片
  平板：双列 grid
  桌面：三列 grid

TripDetailPage:
  手机：堆叠布局
  桌面：Hero 全宽 + 内容 max-w-3xl 居中

所有页面：
  min-h-dvh + pb-14（底部导航高度补偿）
```

### 4.3 移动端安全区

```css
/* BottomNav 底部 safe-area 适配 */
.pb-safe {
  padding-bottom: env(safe-area-inset-bottom, 0px);
}
```

## 五、文件结构

```
frontend/src/
├── components/            # 通用 UI 组件（14 个）
│   ├── Button/  Card/  Input/  Tag/  Toast/  Modal/

│   ├── auth/              # LoginPage + MePage
│   ├── trips/             # TripListPage + TripDetailPage
│   └── map/               # MapView + SvgRouteMap
├── hooks/                 # useAmap / useChatStream / useAuth / useMediaQuery
├── services/              # API 封装（fetch + JWT 注入 + Token 刷新队列）
├── stores/                # Zustand stores（authStore / chatStore）
├── design-tokens/         # Design Token 常量
├── App.tsx                # 路由配置
├── main.tsx               # 入口
└── index.css              # 全局样式 + 天气背景 + 粒子 + 动画
```
