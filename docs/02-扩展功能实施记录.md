## 概述

在初始 4 工具骨架基础上，实施了 5 项扩展：高德地图 API、高德 POI 搜索、BGE 向量检索、Token 追踪、酒店搜索。所有扩展均已完成编码并通过测试。

---

## 扩展1：高德地图 API 真实路线规划

**文件**：[src/tools/amap_service.py](../src/tools/amap_service.py)

`AmapService` 封装高德 Web API：

| 方法 | API 端点 | 功能 |
|------|----------|------|
| `geocode()` | `/v3/geocode/geo` | 地名 → 经纬度 |
| `get_distance()` | `/v3/distance` | 驾车距离 + 耗时 |
| `plan_route()` | 组合调用 | 完整路线规划（含交通方式） |
| `search_poi()` | `/v3/place/around` | 周边 POI 搜索 |
| `search_restaurants()` | 委托 `search_poi()` | 周边餐厅 |
| `search_hotels()` | 委托 `search_poi()` | 周边酒店 |

**降级策略**：高德 API → Fallback 坐标 + Haversine 公式 → 错误提示

## 扩展3：BGE Embedding 向量检索升级

**文件**：[src/rag/embeddings.py](../src/rag/embeddings.py) + [src/rag/retriever.py](../src/rag/retriever.py)

| 优先级 | 后端 | 模型 | 说明 |
|--------|------|------|------|
| 1 | FlagEmbedding | BAAI/bge-small-zh-v1.5 | 最快 |
| 2 | sentence-transformers | BAAI/bge-small-zh-v1.5 | 通用 |
| 3 | TF-IDF | 字符级 n-gram | 零依赖降级 |

混合评分：`final_score = 0.3 × BM25 + 0.7 × vector_similarity`

---

## 扩展4：Token 用量追踪与成本估算

**文件**：[src/server/token_tracker.py](../src/server/token_tracker.py)

| 类 | 功能 |
|---|------|
| `TokenUsage` | 单次调用 token/cost 数据类 |
| `SessionStats` | 会话维度聚合统计 |
| `TokenTracker` | 全局追踪器（记录 + 统计 + 报告） |

DeepSeek 定价：输入 ¥1/1M tokens，输出 ¥2/1M tokens。支持预算告警和日/周成本报告。

---

## 扩展5：酒店预订工具

**文件**：[src/tools/hotels.py](../src/tools/hotels.py)

支持按城市、附近景点、预算、入住/离店日期搜索住宿。数据来源：高德 POI（type=080000 住宿）→ Mock fallback（10 条）。自动计算住宿天数 × 每晚价格。
